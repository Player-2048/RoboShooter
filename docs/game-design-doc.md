## 一、游戏概述

**游戏类型：** 搜打撤（Search-Fight-Extract）第一人称射击游戏

**核心设定：** 玩家扮演远程操控者，在安全地点通过数据传输链路遥控**机器人载体**进入危险区域执行任务。可控核聚变已普及，能源不再稀缺；真正的稀缺资源是**材料和加工能力**。

**与同类游戏的关键差异：**

| 传统搜打撤 | RoboShooter |
|-----------|-------------|
| 人类干员，死了就是死了 | 机器人载体，损毁=装备丢失，操作者安全 |
| 单一血条 | 双血条：机械耐久 + 信号强度 |
| 局外整理装备 | 局内随时管理仓库、交易、加工 |
| 固定地图认知 | 动态地图建模，永久解锁，下局保留 |
| 个人仓库独立 | 团队共享背包空间，协作收益 |
| 无算力概念 | 算力池共享，黑客攻防，队友可操控你的载体 |

## 二、核心对局循环

### 2.1 战前阶段

1. **选择地图** — 矿场 / 维和地图
2. **匹配队友** — 3人或4人（可能同时开放两种模式）
3. **准备空间（更衣间）** — 与队友商谈阵容：每个人选什么机器人、什么装备配置
4. **部署进入** — 进入目标地图

### 2.2 局内阶段

- 搜刮物资、战斗、完成任务
- 可随时访问**内置论坛**查阅攻略/情报
- 可随时访问**商店**预购下局装备
- 可随时进行**装备维修、物品合成、材料熔炼、拆解**
- 可**预出售**物资（撤离前锁定交易）

### 2.3 撤离

- **多个撤离点** + 多类型撤离点
- **强制倒计时**：地图存在基于通讯频率的信号干扰，随时间推移逐渐增强
- 倒计时结束时玩家**彻底断联**——所有物资和装备全部丢失
- 撤离不需要全队一起：可牺牲队友拖住敌人，主力带关键物资走
- 撤离后收益**事后分配**

### 2.4 机器人损毁

- 机器人本质上是玩家带进对局的装备，损毁 = **永久丢失**
- 经济崩盘时提供**最基础廉价机器人**：
  - 性能极低
  - 用该机器人带出的物资会被强制扣除一部分作为购买成本
  - 可理解为"额度无限、先用后付"

## 三、经济与成长

### 3.1 核心货币

- **资源本位制**：货币绑定矿石和制造蓝图
- 可在游戏内银行/机构用恒定价格兑换材料
- 无删档机制

### 3.2 加工能力变现

- 不同玩家发展程度不同 → 加工能力差异
- 低加工能力玩家想要高端装备 → 付费使用高端加工设备
- 费用以绑定资源的货币支付

### 3.3 获取装备路径

| 方式 | 说明 |
|------|------|
| 购买原材料 | 在局内/局外商店购买 |
| 局内获取原材料 | 搜刮带出 |
| 加工零件 | 原材料 → 零件（需要蓝图） |
| 购买零件 | 直接购买，跳过加工 |
| 组装成品 | 零件 → 机器人/武器 |
| 蓝图解锁 | 解锁对应零件的制造方法（矿石→金属锭→特定流程→成品） |

### 3.4 交易与安全

- 加工器械和场地可出售（方便局外交易）
- **二级仓库**：防止借号后被恶意毁号
- 核心资产放入二级仓库需额外验证

### 3.5 单人 vs 组队

- 收益本质相同（都是带出物资）
- 组队优势：**共享背包空间**，多一个人 = 多一份容量 = 每人多一份收益
- 设计目标：消除因物资背刺队友的动机
- 待解决问题：消极怠工（不干活只分物资）的防范机制

## 四、血条与生存系统

### 4.1 机械耐久度（血量条）

- 代表机器人躯体的物理完整性
- 被打光 → 机械躯体无法运作
- 显示方式：人型机械二维平面图，掉血=降水位（类似 CF 爆破模式人像血条）
- 考虑提供多种血条样式供玩家选择

### 4.2 信号强度（信号条）

- 玩家远程控制机器人的数据传输链路质量
- 被打光/被干扰至归零 → 玩家与载体断联 = 死亡
- 显示方式：滚动斜棱形长方框，绿白交替配色，降低时变黄→变红
- 信号干扰来源：时间推移（机制）、敌方黑客攻击（主动）

### 4.3 能量系统（技能蓝条）

- 使用技能消耗能量
- 能量不足可能导致：无法使用技能、部分功能受限
- 可通过道具补充
- 显示：屏幕中下，黄色（类似机油颜色）

### 4.4 伤害反馈

- **受击推算方向**：机器人躯体受力感应 → 反向推算弹药来向（非小地图显示）
- **被击中音效**：金属撞击声（金属弹头 vs 金属躯体）
- **命中反馈**：敌方被击中飚出金属碎屑（类比人类爆血雾）

## 五、机器人系统

### 5.1 三种机器人类型

**人形机器人：**
- 参考宇树机器人外形 + 核动力
- 全能型，无明显短板
- 可装载集成型载荷

**异形机器人：**
- 本质人型 + 外置模块
- 赛博朋克机械异肢风格
- 更多技能槽位

**关节分布式机器人：**
- 灵感：钢铁侠战甲
- 紧急时分离解体飞散逃离，之后再组装
- 可附身 AI 机器人实现敌方控制
- 专属技能：分离逃脱

### 5.2 强度与差异

- **无轻中重之分**，可互相转化
- 机器强度 = 使用什么部件（高性能部件=高耐久+低重量）
- 差异体现在：技能槽数量、背包容量、专属功能
- 取决于玩家的对局策略，而非固定职业

### 5.3 武器通用性

- 所有机器人通用所有武器（类似人类手持武器）
- 差异在于身体性能

### 5.4 外置模块（技能）

- 勾爪、爬墙、喷射弹跳等
- 玩家自选，受技能槽限制
- 可理解为"玩家自选技能"

### 5.5 外观

- 可修改表面图案（贴图、喷剂）
- 不能完全自定义形状
- 机器人皮肤：运营层面待定

### 5.6 维修

- 部件损坏 → 拆下维修或直接更换
- 随身携带备件 → 打架损坏快速替换恢复血量
- **维修套件**：整合各零部件，可现场生成所需部件
  - 本质是小型零件熔炼组件
  - 消耗对应材料
  - 可加入材料补充消耗
- 也可用电焊枪、热熔塑性机现场修复

## 六、枪械系统

### 6.1 定装火药武器

- 传统枪械原理
- 装填子弹 → 激发底火 → 发射弹头
- 子弹耗尽后更换弹夹
- 弹夹装载量系统

### 6.2 电磁弹射武器

- 充能进入激活状态
- 将弹夹内异形金属弹头加速发射
- 弹头类型效果：

| 弹头类型 | 效果 |
|---------|------|
| 金属射流弹 | 穿透性强 |
| 化学腐蚀弹 | 腐蚀性金属，持续伤害 |
| 强磁碎屑弹 | 命中碎裂附着机体，叠加电磁干扰，附着越多干扰越重 |

### 6.3 武器 UI 展示

- 右下角枪械 + 子弹数量（参考 Apex Legends）

## 七、算力与黑客系统

### 7.1 算力消耗行为

- **入侵破解**：类似三角洲"房卡"但更灵活
- 部分破解强相关特殊道具（无道具则无法破解）
- **激活场地设备**：短暂控制场景内强大道具
- **激活全图警戒系统**
- **侵入 AI 机器人**：制造迷惑行为、传递虚假信息、制造陷阱

### 7.2 信号受攻击反馈

- 画面噪点（数据回传受影响）
- 某些功能被禁用或效果降低
- 信息功能短时间内被 ban

### 7.3 防御黑客

- 本质：增加算力消灭攻击行为
- 多数黑客攻击效果 = 短时间干扰 + 算力消耗
- 反向入侵极困难，通常只能抵御

### 7.4 算力池共享

- 每个人自带算力设备决定局内算力规模
- 开局可选择将多少算力加入**共享池**
- 共享池算力：队友无需过问直接使用
- 无距离限制（同一数据传输链路）
- 极端情况：后期可直接**控制队友的载体**、使用他的装备
- 队友载体死亡后：观战视角 → 帮队友操作面板/破解系统（副驾驶模式）

### 7.5 算力 UI

- 屏幕中下位置（与能量条并列）
- 高亮光、微弱数字特效

## 八、地图与探索

### 8.1 矿场地图

矿场是一个**全自动化生产生态系统**，10 个设施构成从开采→运输→加工→寄出的完整产业链。以下为每个设施的详细设计。

---

#### 设施 1：采矿机

**功能定位**：矿场生态的起点，持续自动运转的大型挖掘机械。

**运作机制**：
- 自动开采原矿，通过运输链路送往熔炼站
- 本身全自动运转，不需要玩家干预
- 产出物（原矿）会堆积在出料口等待运输车或传送带运走

**玩家交互**：
- **破坏获取零件**：攻击采矿机可破坏机械部件，获得损坏的零件（可拆解为材料）
- **截获矿石**：在运输链路上拦截正在运送的原矿，装入背包
- 采矿机本身不可操作、不可入侵、不可关闭

**战术价值**：低风险低收益的基础物资点，适合开局阶段快速收集材料

---

#### 设施 2：矿物运输车

**功能定位**：运送高价值物资的重型全地形自动运输载具，附带 AI 护卫。

**运作机制**：
- 固定路线、全地形适应，运送高价值物品（不限于矿物——常见类型包括：高价值金属锭、稀有电子元器件、制造材料）
- 路线被障碍物阻断或被破坏时，自动重新规划路径
- 固定有相当数量的 AI 哨兵机器人沿路护送

**玩家交互**：
- **破坏截停**：攻击运输车可截停，获得车上装载的高价值物资
- 触发 AI 哨位警报 → 引发"劫掠战"，大量 AI 哨兵增援
- 运输车不可操作、不可劫持改道、不可伪装混入
- 后续考虑加入**护送任务**：玩家接取任务后，现场开启运输口派出一辆运输车，护送成功获得物资回报

**战术价值**：高风险高回报的"移动战利品箱"，需考虑能否承受截停后的 AI 增援压力

---

#### 设施 3：熔炼站

**功能定位**：物资压缩打包点——将低价值原矿熔炼为高价值金属锭。

**运作机制**：
- 接收采矿机产出的原矿，通过高温熔炉炼制金属锭
- 自动运转，属于矿场生产链的第二环节
- 熔炼完成的金属锭暂存于冷却架上

**玩家交互**：
- **物资压缩**：玩家将背包中的低价值矿石（占4格）熔炼为金属锭（占1-2格），腾出背包空间携带更多物资
- **搜刮成品**：直接搜刮冷却架上已完成的金属锭
- 熔炼过程需要一定时间，不可加急
- 不可关闭、不可入侵

**战术价值**：中低风险中低回报的资源利用点，提高单局物资携带效率

---

#### 设施 4：加工中心

**功能定位**：金属锭→零件→装备的精密制造设施，可合成稀有材料。

**运作机制**：
- 全自动机械臂流水线，将金属锭加工为零件，或将零件组装为成品装备
- 24小时运转，设备磨损需要维修零部件储备（→ 储物间）

**玩家交互**：
- **零件制造**：将金属锭加工为武器/机器人零件
- **装备组装**：将多个零件组装为成品装备
- **特殊合成**：用多种复合材料制作高性能合金（加工时间长，所需材料稀有，产出超高性能稀有材料）
- **稀有产品制造**：通过入侵/激活设备，将普通材料转化为局外无法购买也无法自制的特殊产品
- **搜刮储物间**：加工中心配套的设备储物间存放大量加工设施备用配件，是高级搜刮点
- **拆解回收**：拆解捡到的破损零件，送往熔炼站转化为金属锭再重新加工
- 加工需要耗时，不可加急

**战术价值**：高风险（入侵触发警报）高回报的精锐装备制造点

---

#### 设施 5：寄存站

**功能定位**：局内物资即时结算+邮件配送点，促使玩家之间竞争的核心设施。

**运作机制**：
- 矿场内外界物资转运枢纽——矿石→金属锭→零件→成品都通过这里寄出
- **每局随机性开启**：不是每局都开放，开放数量不定
- 固定地图位置

**寄存功能**：
- 将背包中物资直接结算寄出——哪怕玩家之后死亡，寄存物资也算入收益
- 存入后立刻生效但进入**守卫阶段**（几分钟），期间无法取出
- 寄存容量很大（足够装一局全部收益），队伍成员共享容量
- 寄存守卫期间，**全图所有玩家地图上显示该寄存站位置**
- 寄存结束后该站点本局永久关闭
- 守卫期间其他玩家可进攻抢夺寄存物资，杀人越货并重新刷新守卫计时

**配送功能**：
- 消耗货币/资源呼叫加工中心的加工服务，购买"材料+加工"得到指定装备成品
- 装备由寄存无人机运送，1-2分钟内空投到指定位置
- 这也是"寄存站"名称的含义——矿场的物流枢纽

**触发效果**：
- 开启寄存即触发全图 AI 警戒，哨兵向寄存站发动清除行动

**战术价值**：超高收益、超高风险的"双倍物资"机会，强制 PvP 和 PvE 对抗

---

#### 设施 6：能源中心

**功能定位**：不可摧毁的全局供电枢纽，玩家能量补给站，干扰战术节点。

**运作机制**：
- 为整个矿场所有设施和 AI 哨兵提供能源
- 产出各类能源产品（能量电池、能源燃料），适配不同工作环境

**玩家交互**：
- **能量补给**：补充机器人能量（技能消耗资源）
- **搜刮能源产品**：带走不同类型的电池/燃料作为物资
- **链路干扰**：干扰能源中心到各设施的能源传输链路，引发以下效果：
  - 某设施运转暂停/效率降低
  - 人为触发警报，提升全场 AI 哨兵警戒等级
- **不可关闭、不可摧毁、不可入侵**

**战术价值**：基础设施 + 能量补给站 + 战术干扰点

---

#### 设施 7：地下交通

**功能定位**：矿场最左端到最右端的地下直线通道，高风险快速移动路线。

**结构**：
- 只有两个出入口：矿场最左端和最右端
- 地下通道直线连接，长度覆盖全图
- 通道内无任何其他内容——没有敌人、没有物资、没有分支

**入口防御**：
- 两端口各有一座**无限补充型哨兵军械库**
- 正常情况下：铁门封锁，哨兵静默
- 高警戒状态下：哨兵无限生成，摧毁一个立刻补充一个

**使用方式**：
- 玩家可通过**数据入侵**获得短暂身份伪装许可，不惊动哨兵穿越
- 也可以硬闯（不推荐——无限哨兵补充）

**地图建模限制**：
- 地下区域几乎无法扫描建模（无法在运输途中下车交互）

**战术价值**：跨图快速转移通道 + 从A到B的战术捷径，但身份伪装失败 = 无尽战斗

---

#### 设施 8：中央控制室

**功能定位**：由 AI 智脑掌控的矿场终极核心，地图最难以攻破的目标。

**防御等级**：
- 地图最高级别防守：层层军械库 + 大量门口门禁
- 需要多种条件组合才能获得入侵机会：
  - 算力破解门禁
  - 黑客入侵安全系统
  - 特定场地事件触发
  - 其他队伍为方便而开启的通路被利用

**入侵效果**：
- **触发最高级别全图警报**
- **短暂获取矿场信息**（某设施状态、地图数据等）
- **短暂干扰某设施运转**
- AI 智脑检测到入侵后强制重启整个系统 → 一段空窗期 → 空窗期结束后**所有玩家的黑客入侵伪装和破译全部失效**

**入侵收益**：
- 剧情道具获取
- 制造工艺等级提升（必须完成此挑战才能升到下一级）
- 没有高价值物资掉落，没有场地武器控制权

**战术价值**：地图终局挑战，不是物资点而是"资格证"——证明玩家对地图和实力的理解已经到达顶级

---

#### 设施 9：机械维修站

**功能定位**：全矿场机械设备的后勤保障，玩家耐久度恢复+改装点。

**运作机制**：
- 全矿场所有机械设备维修零部件的集中存放地
- 各种损坏的机器设备和替换备件在这里汇集

**维修方式**：

| 模式 | 条件 | 说明 |
|------|------|------|
| 自动维修 | 需通过黑客入侵获取矿场机器身份识别 | 彻底修复无法运行的设备，清理机器人将损坏的机械运来这里修复 |
| 自修理 | 无需条件 | 玩家机器人进入维修间，像换衣服一样更换自身损坏部件 |

**物资搜刮**：
- 高级机器人的库存备用件——稀有外置模块（高价值装备）
- 破损但可分解的电子元器件——带回局内拆解为小型材料
- 大部分零件破旧且多次使用过，不如加工中心的备用件品质高
- 可补充机械维修套件消耗材料

**与加工中心的区别**：
- 加工中心：全新零件、精密制造、高级合金合成
- 机械维修站：破旧替换件、拆解回收、损坏零件的二次利用

**战术价值**：后勤保障 + 回血点 + 稀有外置模块搜刮点

---

#### 设施 10：哨兵军械库

**功能定位**：惩戒性质的军备设施，维持地图平衡的"软边界"。

**分布**：
- 分散在各关键设施附近
- 需要约束玩家行为、让玩家有所顾虑的地方都会部署

**类型与激活状态**：

| 状态 | 哨兵数量 | 补充方式 | 触发条件 |
|------|---------|---------|---------|
| 静默 | 0 | — | 默认状态，铁门封锁 |
| 低警戒 | 1-2名巡逻 | 被摧毁后一小段时间唤醒新哨兵填补 | 轻微违规行为 |
| 高警戒 | 无上限 | 摧毁即补，无延迟，无上限 | 严重入侵/攻击运输车/开启寄存站等 |

**关键限制**：
- 哨兵武器的零件有专门标记——携带领到的零件会被全图警戒哨兵透视锁定
- 无法偷取军械库武器
- 无法击杀哨兵搜刮物资（零件追踪机制导致不可用）
- 可以短暂入侵瘫痪军械库——但入侵结束后该军械库直接进入高警戒状态
- **无法进行长线 AI 消耗战**——想靠刷哨兵拖延是不可行的

**战术价值**：地图平衡工具，不是资源点——迫使玩家规划行动路线，权衡冒险价值

### 8.2 推荐新增设施（AI 建议）

基于对10个已有设施的分析，矿场地图当前缺失以下几类功能。以下推荐供你评估：

---

#### 推荐 1：信号中继塔

**理由**：你的游戏核心机制之一是"信号强度"。目前信号只衰减不补充——缺少一个让玩家能主动维护信号水平的地点。

**功能设想**：
- 地图上有2-3座信号中继塔，分布在矿场不同区域
- 玩家可爬塔操作控制台，**临时增强附近区域的信号强度**——抵消信号衰减、抵抗黑客干扰
- 塔顶是狙击制高点，天然具有战术价值
- 可以入侵中继塔放大自己的算力范围
- AI 哨兵也会巡逻防御——因为这是矿场关键通信基础设施

**跟已有设施的呼应**：与中央控制室的 AI 智脑通信链路相连；与哨兵军械库的警戒信号联动——摧毁中继塔会削弱该区域的警戒补充速度

---

#### 推荐 2：废料处理场

**理由**：矿场全自动生产，一定有废弃物。目前熔炼站和加工中心产出高质量材料，但没有"回收利用"的渠道。

**功能设想**：
- 矿场边缘区域的大型垃圾倾倒场/废料堆
- 可以找到**带损坏标记的低价值零件**——不能直接使用，但可以拆解回收
- 偶尔混有被丢弃的损坏精密电子元件（稀有材料来源）
- 环境风险：不稳定废料堆可能塌陷，某些废料有微弱辐射，需要机器人有相应防护模块才能进入深处
- 没有 AI 重兵把守——这里对矿场来说就是垃圾

**跟已有设施的呼应**：废料送来机械维修站筛选，还有价值就修修再用，没价值的电子元件可以拆解回炉

---

#### 推荐 3：矿场员工居住区

**理由**：矿场是全自动的，但设施维护、系统监控、紧急故障处理仍然需要人（或至少是远程操控者的本地代理人）。这解决了"谁在这里工作"的世界观问题。

**功能设想**：
- 废弃的居住模块——食堂、宿舍、医疗室、娱乐室
- 可以搜刮到个人物品：数据盘（含造工艺蓝图碎片）、个人终端（含非加密的内部邮件/日志→世界观叙事）
- 没多少军械库防御——这里不是高价值战略区域
- 环境叙事道具：墙上的排班表、食堂菜单、某工人的维修日记——这些书写写好了可以让地图"活起来"
- 地下通道可能有一个出入口连接到这里

**跟已有设施的呼应**：员工曾负责监控采矿机/熔炼站的运转，他们的日志能间接透露地图机制信息给新玩家

---

#### 推荐 4：样品实验室

**理由**：加工中心做的是量产加工，但矿石中偶尔会发现异常材料需要单独分析。这是矿场的"质检部门"。

**功能设想**：
- 小型分析实验室，存放被送来做成分检测的稀有矿石样品
- 可以找到**高纯度稀有矿石**——数量少但价值极高
- 实验室设备可以用来**分析未鉴定的蓝图碎片**（将"未知蓝图碎片"鉴定为具体蓝图）
- 进入需要破解或高算力消耗
- 地处偏僻但有少量哨兵巡逻

**跟已有设施的呼应**：发现异常材料的样品会被送到这里检测；发现新型合金配方时通知加工中心

---

#### 推荐 5：废弃撤离点

**理由**：你提到了多个撤离点——除了地下交通的特殊撤离点外，主撤离点怎么设计？

**功能设想**：
- 地图上有3-4个撤离点，不是全部可用——每局随机激活其中一部分
- 其中一个可以是"紧急撤离点"——单人可用，但随机有更高信号干扰风险
- 撤离点附近是 PvP 高发区——大家都往这里赶
- 某个废弃撤离点可能有隐藏的物资箱——因为上一批撤离的玩家在匆忙中落下的

**跟已有设施的呼应**：撤离点的信号干扰等级与能源中心的链路状态、信号中继塔是否完好有关

## 九、动态地图建模

### 9.1 核心机制

- 玩家在特定区域内开启**扫描模式**并待一段时间
- 参考现实：录制场地图像 → 反向建模场景
- 特别区域有强屏蔽干扰 → 需完成任务/破解 → 直接复制建模信息 + 获取交互权限（门禁、特殊功能等）

### 9.2 建模效果

- 打开上帝视角 → 剥离贴图、只留关键线条的粗糙空间模型（类似 CS2 Dust2 白模）
- 可缩放、旋转
- 已建模区域**永久解锁**（类似空洞骑士买地图 + 探索点亮）
- 每解锁一个区域 = 建模空间中亮起一个区域

### 9.3 场景状态记录

- 玩家走过已建模区域 → 场景交互状态自动记录
- 再次经过时对比记录 → 有变化 = **有人来过** → 地图标红
- 队友分开走 → 队友路径自动为自己更新
- 发现队友路径场景变化 → 立即警戒

### 9.4 标记功能

- 在建模地图中标记地点/物资点
- 标记同步到第一人称视角 → 游戏内对应位置显示图标
- 容器标记 + 场景标记
- 与队友共享

### 9.5 设计参考

- 空洞骑士：购买地图 → 探索 → 点亮区域
- 猎杀对决：玩家推理"有人来过"
- 本系统：自动化+持久化+可分享

### 12.2 协作机制

## 十三、音频设计

### 13.1 材质音效

- 多材质地面：金属、泥土、碎石等
- 不同型号机器人走路声音可能不同
- 用于判断：什么类型的机器人在什么地面上、走了多远

### 13.2 信号干扰音效

- 数据传输受干扰时有杂音
- 但玩家终端可预处理数据，大幅隔绝大部分杂音影响
- 所以不会很严重，只是提示性存在

### 13.3 机械音

- 机器人自身运行机械音**不做传播效果**
- 理由：游戏中不只是玩家机器人，声音太杂；类似 CS2 切枪无声

### 13.4 战斗音效

- 金属撞击声（子弹命中机器人）
- 金属碎屑飙出音效
- 机器人移动时伺服/液压声（不给敌方听到）
