# RoboShooter（暂定）
一款远程操控机器人对战的第一人称射击游戏。

## 当前状态
极早期开发中，核心玩法原型尚未完成。

## 我想做什么
- 玩家远程控制各类机器人，而非人类角色
- 设想中的未来风格武器
- 快节奏的多人枪战体验

## 技术栈（准备用的）
- 游戏引擎：Unity（例子）
- 编程语言：C#
- 版本控制：Git & GitHub

## 如何参与
目前项目由我个人学习开发，欢迎任何形式的交流与建议。
如果你有兴趣一起玩或者合作，请提 Issue 或发邮件（可选）联系。
